# WanderLust
<b>WanderLust</b> is a travel website that connects you to various other travellers and lets you to view places and create trips to various places.<br>
It also allows you to add friends and create trips with these friends<br>

Steps to Use :<br>
<ol>
  <li>Clone the repository : </li>
  <li>Copy this folder into <b>C:/Xampp/htdocs/</b></li>
  <li>Start the Xampp Control Panel and Start the Apache Server and SQL Server</li>
  <li>Visit <b>localhost</b></li>
